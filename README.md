## Purpose of JAR Folder

The JAR folder contains the  `.jar` file which is reponsible
 for running this project.  

## Required Information when Submitting Tank Game
Version of Java Used:** Java 21 (OpenJDK 21).


IDE Used:** IntelliJ IDEA.

Steps to Import Project into IDE
---
First step to take is cloning the project rep from GitHUB to computer. Then, open Intellij IDEA, doing File → Open and chose the root folder of the project. Select correct Java 21 is selected


***Steps to Build Project
---
From the top menu, select Build → Build Project to compile the source code. 


To create the runnable JAR file, select Build → Build Artifacts → Build, which will generate TankGame.jar in the JAR/ folder.

***Steps to Run Project
---
Download the repository as a ZIP from GitHub and unzip it.


On terminal: cd ~/Downloads/tankgame--janvi-kumar-main


Command to run game:java -jar jar/TankGame.jar


Then Select Level and press start.



***Controls to play your Game
---
| Action | Player 1 (Red Tank) | Player 2 (Blue Tank) |
|--------|---------------------|----------------------|
| Move Forward | W | Up Arrow |
| Move Backward | S | Down Arrow |
| Rotate Left | A | Left Arrow |
| Rotate Right | D | Right Arrow |
| Shoot | Space | Enter |
